Этот гайд описывает подключение к **Shadowsocks**.

**Shadowsocks** — это не VPN, а **прокси-протокол**, предназначенный для:
- маскировки трафика;
- работы в сетях с жёсткой фильтрацией.

> Рекомендуется только для продвинутых пользователей, в остальных случаях по возможности используйте другие клиенты.

---

## Что понадобится

Для подключения к Shadowsocks тебе понадобятся:

- **адрес сервера**
- **порт**
- **пароль**
- **метод шифрования**, в нашем случае - `aes-256-gcm`
- **plugin / plugin options**, в нашем случае - `obfs-local` с опциями: `obfs=tls;obfs-host=cloudflare.com`

Чаще всего всё это передаётся одной строкой `ss://` или QR-кодом

## Android

### Вариант 1. Shadowsocks (официальный клиент)

1. Установи **Shadowsocks** из Google Play: https://play.google.com/store/apps/details?id=com.github.shadowsocks
2. Открой приложение
3. Добавь сервер:
   - через QR-код  
   - или вручную
4. Заполни параметры:
   - Server
   - Port
   - Password
   - Encryption method
5. Включи прокси

## Режимы работы (важно)

Большинство клиентов поддерживают несколько режимов:

- **Global** — весь трафик устройства проходит через прокси  
- **PAC / Rule-based** — только выбранные сайты  
- **Manual** — прокси на уровне браузера

Для повседневного использования обычно хватает **PAC / Rules**.

Удобно для использования совместно с Proxifier.
